归并排序是将一个序列不断的一分为二,直到还是一个元素就变成了有序.然后将两个有序的部分数组进行合并,合并到最后得到的就是一个有序的序列.
# 算法实现

```python
MergeSort(list):
	MergeSort(list[0,len(list)//2])
	MergeSort(list[len(list)//2+1:])
	merge(list,0,len(list)//2,len(list)-1)
```

根据上面的算法实现,可以看出来归并排序是典型的分治算法.不断的分为两部分,然后进行有序数组的合并

# 时间复杂度
由于每一层分为两部分.所以整棵树的高度是 $log_2n$.每一层的归并的消耗的时间复杂度是O(n).所以归并排序的时间复杂度是O(n$log_2n$)
# 使用场景
归并排序非常适合使用的场景是链表的排序.因为在每一个层次的归并过程中,需要一个额外的空间俩存放中间的归并结果,这里设计到大量的数据copy.而链表就没有大量链表需要拷贝的问题.
当然对于链表使用归并排序,这里会带来一个问题是对链表进行分割,这里使用一个快慢指针,这样慢指针就指向了中间的指针.
