# 结构
优先级队列就是一种对元素进行优先级排序的容积.提供了写入,查询优先级最高元素,和取出优先级最高元素的操作.
# 操作
优先级队列中提供的操作:
```c
void insert(Q `*q`,item_type key);
item_type find_min(Q `*q`);
item_type del_min(Q `*q`);
```

# 性能
优先级队列这种操作,底层选用的数据结构有三种,一种是数组,有序数组,平衡二叉树.
这三种操作的性能是不一样.具体如下:

|                | 无序数组     | 有序数组     | 平衡二叉树    |
| -------------- | -------- | -------- | -------- |
| Insert         | O(1)     | O(n)     | O(log n) |
| Find-Minimum   | **O(1)** | **O(1)** | **O(1)** |
| Delete-Minimum | O(n)     | O(1)     | O(log n) |

# 概念
优先级队列在很多场景下都使用了.例如top-k问题,最短路径等场景下.
