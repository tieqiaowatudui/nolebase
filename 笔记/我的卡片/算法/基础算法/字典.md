# 结构
字典是一种按照key值获获取值的容器结构.写入数据的时候将key和value写入到字典中,读取出来的时候也是按照key获取value.删除也一样的按照key删除.高级的字典还能返回某个key的前一个元素,或者后一个元素.返回当前字典中最大的元素和最小的元素.
# 操作
字典主要支持了查询,写入,删除,前一个,后一个,最大和最小这七个操作.
- search(key)
- insert(key)
- delete(key)
- successor(key)
- predecessor(key)
- min()
- max()
# 性能
对于字典向外提供的功能,底层不同的实现方式得到的性能也不同.可以使用数组实现,也可以使用单向链表和双向链表实现.对于数组和链表也有有序和无序这两种实现.相对应的实现字典这里能够选择的底层数据结构就是6中.这6中不同的数据结构,对于不同的操作的时间复杂度是有差别的.不可能满足所有的优势.

| 字典操作              | 无序数组  | 有序数组     |
| ----------------- | ----- | -------- |
| Search(L, k)      | O(n)  | O(log n) |
| Insert(L, x)      | O(1)  | O(n)     |
| Delete(L, x)      | O(1)* | O(n)     |
| Successor(L, x)   | O(n)  | O(1)     |
| Predecessor(L, x) | O(n)  | O(1)     |
| Minimum(L)        | O(n)  | O(1)     |
| Maximum(L)        | O(n)  | O(1)     |
| 操作                | 单向无序  | 双向无序 | 单向有序  | 双向有序 |
| ----------------- | ----- | ---- | ----- | ---- |
| Search(L, k)      | O(n)  | O(n) | O(n)  | O(n) |
| Insert(L, x)      | O(1)  | O(1) | O(n)  | O(n) |
| Delete(L, x)      | O(n)* | O(1) | O(n)* | O(1) |
| Successor(L, x)   | O(n)  | O(n) | O(1)  | O(1) |
| Predecessor(L, x) | O(n)  | O(n) | O(n)* | O(1) |
| Minimum(L)        | O(n)  | O(n) | O(1)  | O(1) |
| Maximum(L)        | O(n)  | O(n) | O(1)* | O(1) |


# 概念
对于技术选项的选择,需要考虑不同不同的操作的支持,如果能够很好的支持 A+B两个操作,那么大概率不能最优的支持A操作,也不能最优的支持B操作.所以技术选项的时候,需要权衡,根据不同操作的使用评率,以获取到最优的选择.
