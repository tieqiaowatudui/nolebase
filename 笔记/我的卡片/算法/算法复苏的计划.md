## 算法学习与LeetCode实践计划

### 第一阶段:基础数据结构(3-4周)

**学习内容:**

- Algorithms 4th: 第1章(基础),第3章(查找)的基础部分
- Algorithm Design Manual: 第3章(数据结构)

**核心知识点:** 数组、链表、栈、队列、哈希表、字符串

**LeetCode推荐题目(20-25题):**

- 数组: Two Sum, Best Time to Buy and Sell Stock, Maximum Subarray
- 链表: Reverse Linked List, Merge Two Sorted Lists, Linked List Cycle
- 栈/队列: Valid Parentheses, Implement Queue using Stacks, Min Stack
- 哈希表: Group Anagrams, Longest Substring Without Repeating Characters
- 字符串: Valid Anagram, Longest Palindrome, Reverse String

### 第二阶段:排序与搜索(2-3周)

**学习内容:**

- Algorithms 4th: 第2章(排序)
- Algorithm Design Manual: 第4章(排序与搜索)

**核心知识点:** 快速排序、归并排序、堆排序、二分查找

**LeetCode推荐题目(15-20题):**

- Sort Colors, Merge Sorted Array, Kth Largest Element
- Binary Search, Search in Rotated Sorted Array
- Find First and Last Position, Search a 2D Matrix
- Meeting Rooms II, Top K Frequent Elements

### 第三阶段:树与图基础(4-5周)

**学习内容:**

- Algorithms 4th: 第3章(二叉搜索树),第4章(图)
- Algorithm Design Manual: 第5章(图遍历),第7章(图问题)

**核心知识点:** 二叉树、BST、DFS、BFS、最短路径

**LeetCode推荐题目(25-30题):**

- 二叉树: Maximum Depth, Invert Binary Tree, Symmetric Tree
- BST: Validate BST, Lowest Common Ancestor
- DFS/BFS: Binary Tree Level Order, Number of Islands, Clone Graph
- 图算法: Course Schedule, Network Delay Time, Cheapest Flights

### 第四阶段:递归与动态规划(5-6周)

**学习内容:**

- Algorithm Design Manual: 第8章(动态规划),第10章(如何设计算法)
- Algorithms 4th: 相关章节补充

**核心知识点:** 递归思维、记忆化搜索、经典DP模型

**LeetCode推荐题目(30-35题):**

- 基础递归: Climbing Stairs, Fibonacci Number
- 一维DP: House Robber, Longest Increasing Subsequence
- 二维DP: Unique Paths, Longest Common Subsequence, Edit Distance
- 背包问题: Partition Equal Subset Sum, Coin Change
- 股票问题: Best Time to Buy and Sell Stock系列

### 第五阶段:贪心算法与回溯(3-4周)

**学习内容:**

- Algorithm Design Manual: 第9章(棘手问题),第11章(NP完全问题)
- Algorithms 4th: 相关补充

**核心知识点:** 贪心选择、回溯框架、剪枝优化

**LeetCode推荐题目(20-25题):**

- 贪心: Jump Game, Gas Station, Minimum Number of Arrows
- 回溯: Permutations, Combinations, Subsets
- N-Queens, Word Search, Palindrome Partitioning

### 第六阶段:高级专题(4-5周)

**学习内容:**

- Algorithms 4th: 第5章(字符串),第6章(背景)
- Algorithm Design Manual: 第12-15章(高级数据结构)

**核心知识点:** 字典树、并查集、线段树、高级图算法

**LeetCode推荐题目(25-30题):**

- Trie: Implement Trie, Word Search II
- Union Find: Number of Connected Components, Redundant Connection
- 拓扑排序: Course Schedule系列
- 高级DP: Burst Balloons, Regular Expression Matching

### 学习建议

**每日时间分配:** 理论学习1-1.5小时 + LeetCode练习1.5-2小时

**学习方法:**

1. 先读书理解算法原理,做书中练习题
2. 每学完一个知识点,立即刷对应的简单和中等题
3. 第一遍不会的题要反复做,间隔几天再做一次
4. 每周末复习本周内容,重做错题
