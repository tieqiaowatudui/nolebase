# 定义
人脑中的奖励系统.构建了一个`预期 -- 行为 -- 奖励`的机制,开确定下一次是否需要进行执行.
整体流程如下:
```txt
                        +------------------+
                        |   设定预期目标    |
                        |  (小任务/大行动) |
                        +--------+---------+
                                 |
                                 v
                        +------------------+
                        |     执行动作     |
                        +--------+---------+
                                 |
                                 v
                        +------------------+
                        | 获取实际结果      |
                        +--------+---------+
                                 |
                                 v
                        +------------------------------+
                        | 计算预测误差                |
                        | Δ = 实际结果 - 预期结果     |
                        +--------+---------------------+
                                 |
           +---------------------+---------------------+
           |                                           |
           v                                           v
  +-------------------+                     +--------------------+
  | 预测误差 > 0      |                     | 预测误差 <= 0      |
  | (正误差/惊喜)     |                     | (负误差/无惊喜)    |
  +--------+----------+                     +---------+----------+
           |                                          |
           v                                          v
  +-------------------+                     +--------------------+
  | 多巴胺释放        |                     | 高层价值评估       |
  | (短期动力)        |                     | (Tonic Dopamine)   |
  +--------+----------+                     +---------+----------+
           |                                          |
           v                                          v
  +-------------------+                     +--------------------+
  | 更新下一次预期    |<--------------------| 判断是否继续行动   |
  +--------+----------+                     +---------+----------+
           |
           v
  +-------------------+
  | 执行下一小任务    |
  +-------------------+


```
