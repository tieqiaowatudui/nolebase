# 定义
人脑中的奖励系统的核心.构建了一个`预期 -- 行为 -- 奖励`的机制,开确定下一次是否需要进行执行.当超过了预期,就分泌多巴胺.否则就分泌负多巴胺.
整体流程如下:
```txt
                        +------------------+
                        |   设定预期目标    |
                        |  (小任务/大行动) |
                        +--------+---------+
                                 |
                                 v
                        +------------------+
                        |     执行动作     |
                        +--------+---------+
                                 |
                                 v
                        +------------------+
                        | 获取实际结果      |
                        +--------+---------+
                                 |
                                 v
                        +------------------------------+
                        | 计算预测误差                |
                        | Δ = 实际结果 - 预期结果     |
                        +--------+---------------------+
                                 |
           +---------------------+---------------------+
           |                                           |
           v                                           v
  +-------------------+                     +--------------------+
  | 预测误差 > 0      |                     | 预测误差 <= 0      |
  | (正误差/惊喜)     |                     | (负误差/无惊喜)    |
  +--------+----------+                     +---------+----------+
           |                                          |
           v                                          v
  +-------------------+                     +--------------------+
  | 多巴胺释放        |                     | 高层价值评估       |
  | (短期动力)        |                     | (Tonic Dopamine)   |
  +--------+----------+                     +---------+----------+
           |                                          |
           v                                          v
  +-------------------+                     +--------------------+
  | 更新下一次预期      |<--------------------| 判断是否继续行动   |
  +--------+----------+                     +---------+----------+
           |
           v
  +-------------------+
  | 执行下一小任务    |
  +-------------------+


```


# 关键点
## 多巴胺问题问题
想要多巴胺每次都发挥作用,其中行动结果 和预期之间的差都是正的.就是每一次都发挥超长.事实上,这种情况是不可能的.这样就会导致动机降低,影响表现,变成负反馈.

## 解决多巴胺无法长久的问题
### 构建新维度的目标
在同一个目标下,如果希望一直超预期的发挥不太现实.这个时候,每一次可以树立不同维度的预期,然后带着这些预期去行动,这样就可以不断获取超预期行动.这样就能不断触发奖励系统,不断提升动机.
