# 结论
大脑对于低收益+困难的工作天然的厌恶.
对于任何复杂一点且有价值的知识都是困难的.
一遍学习   -->  学不会
第二遍重复  --> 没有新东西  --> 浪费
带着浪费 + 模糊不清  --> 消除动机  -->  不想重复第二遍

# 为什么一次学不会
## 本质
学习的本质不是写硬盘,而是修改神经网络的权重
每一次的学习都是修改神经网路的权重.
学习的步骤:
> 1.认识
> 2.区分
> 3.提取
> 4.自动化

所以学习很难一遍完成,并不是将信息写入到脑海中.而是不断的修改脑海中的神经元.这个当中包括:认识 + 区分  + 提取 + 自动化的复杂过程.
学习完一遍的时候,并不是学习完成了,而是知识开始.

## 多学习一遍是否没有用吗?
多学一遍会增加自己的理解感和熟悉感.
但是:
> 理解感  ≠ 学会
> 掌控感 ≠ 会用

我们跟着别人的路径走了一遍信息.
但是这并不是我们自己的路径,就是这些知识没有办法嵌入到我们自己的知识中.

这个过程中:
> 没有反馈,这种反馈不是按照我们期待且能理解的方式出现的
> 


# 为什么不愿意重复一遍

## 重复≠再学一遍
因为没有明确的可以理解的反馈.
我们的重复一遍是无变化的重复
重复一遍以后,得到的反馈是:
> 我看过了
> 我没有新的理解
> 我还是没有办法完全可以使用

低收益 + 高挫败  ==> 不愿意重复
我们不抵制重复  --> 抵制没有收获的重复

# 是什么杀死了动机
对于知识有一些了解,但是不知道如何使用的模糊感
学习了很久,但是没有反馈,或者没有明确的反馈

模糊感(悬半空)  + 没有反馈(不知道到底有没有进步) ==> 杀死动机

# 如何高效重复
每一次都比上一次的重复多一点.更扎实的定位 + 更加可控的反馈.
每次都带着一个问题去重复材料.这样就能够有序的学习知识.
内容没有变 --> 认知在变

# 困难的学习的本质
困难的知识 需要 多轮 重构
优秀的学习者本身就是不断换角度 重构现有知识.

# 思维的转变
一次就能学会的东西要么没有价值,要么就没有办法落地.
每一次重复不是要学习得更加深刻一点,而是是否"获取到更多,更细节的知识?"
