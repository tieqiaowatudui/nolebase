# 困境
在日常生活中,常遇到明明有很多事情需要去做,但是自己就是不想动,磨磨蹭蹭,心里还很难过的情况.    
这种情况的表现形式是知道事情很重要,但是对于事情的认知不清晰,不可预测,不可控导致了自己行动力下降.时间久了,人就会变得压力越来越大,产出越来越低,展现出的形象也越来越弱.进入一个负反馈旋转.  
# 分析
因为对于当前问题的认知的不清晰,不可预测,不可控.这样会导致大脑的负面反应.  
主要包括:
- 不清晰,会导致大脑中需要额外关注多个点,从而导致大脑负荷过高
- 不清晰也会导致大脑对于问题的从开始到最终的路径无法看清,这种情况会导致大脑认为问题无法解决,最终放弃
- 不可预测导致无法启动
