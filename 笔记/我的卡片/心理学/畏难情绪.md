# 困境
在日常生活中,常遇到明明有很多事情需要去做,但是自己就是不想动,磨磨蹭蹭,心里还很难过的情况.    
这种情况的表现形式是知道事情很重要,但是对于事情的认知不清晰,不可预测,不可控导致了自己行动力下降.时间久了,人就会变得压力越来越大,产出越来越低,展现出的形象也越来越弱.进入一个负反馈旋转.  
# 分析
因为对于当前问题的认知的不清晰,不可预测,不可控.这样会导致大脑的负面反应.  
主要包括:
- 不清晰,会导致大脑中需要额外关注多个点,从而导致大脑负荷过高
- 不清晰也会导致大脑对于问题的从开始到最终的路径无法看清,这种情况会导致大脑认为问题无法解决,最终放弃
- 不可预测导致无法启动大脑的奖励系统,就是根据可以预期的结果分泌多巴胺,从而获取到激励反馈,而不可预测会导致过分的悲观
- 不可控意味着风险,而不可控风险会触发大脑的保护机制,最终确定放弃当前任务.

# 解决方案
根据[[#分析]]中确定的问题困境中的内容,我们需要解决问题的,不清晰不可预测和不可控的问题.我们需要解决以上的问题,就能有效的缓解畏难情绪的问题.
- 问题拆解:将一个宏大的复杂的问题进行拆解为小的问题,一直拆解到5秒钟就能够思考清楚的情况.这个时候,整个大的问题已经分解为一个个清晰的小任务了.
- 确定完成标准,什么是完成,什么是完成的好.让问题清洗
- 识别变量,变量包括依赖的外部结果以及产出结果.这些变量哪些是可控的,哪些是不可控的.问题变可控
- 设置反馈点,当前任务中,有哪些点是关键的检查点,这些检查点既是检查点也是反馈点,让整个过程变得预测.
# 一些技巧
用20个字描述清楚当前系统中的关键问题,如果描述不清楚就需要继续拆分.
用5秒钟想清楚当期那任务是做什么的,如果不行,还是继续拆分.
