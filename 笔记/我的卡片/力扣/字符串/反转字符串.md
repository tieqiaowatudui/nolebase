## 6️⃣ 代码实现
```python
from typing import List
class Solution:
    def reverseString(self, s: List[str]) -> None:
        """
        Do not return anything, modify s in-place instead.
        """
        left,right = 0,len(s) -1
        while left < right:
            temp = s[left]
            s[left] = s[right]
            s[right] = temp
            left,right = left+1,right - 1
        
```
