## 3️⃣ 解题思路
### 思路1（数组计数)
- 算法原理：由于全是小写字母,所以使用26位的数组,统计每个字符出现的次数
- 时间复杂度：O(n)
- 空间复杂度：O(1)

### 思路2（用Python3中的Counter数据结构）
- 算法原理：Counter一种特殊的位图,直接比较是否相等
- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 6️⃣ 代码实现
### 手动计数
```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        #异位字符,就是两个字符串中有相同的字母组成的.
        # 首先比较长度
        # 然后使用一个计数法来统计每个数组出现的次数.
        # 
        if len(s) != len(t):
            return False 
        ch_count = [0] * 26
        for ch in s:
            ch_count[ord(ch) - 97] += 1
        for ch in t:
            idx = ord(ch) - 97
            ch_count[idx] -= 1
            if ch_count[idx] < 0:
                return False
        return True
    
```

### 使用Counter类
```python
from collections import Counter
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        # 使用python3中提供的Counter数据额结构.计数的dict
        return Counter(s) == Counter(t)
```
