## 3️⃣ 解题思路
### 思路1
- 算法原理：快慢指针,用跑两步的指针和跑一步的指针,看先跑前面的指针是否能够追上跑得慢的指针.
- 时间复杂度：O(n)
- 空间复杂度：O(1)

## 6️⃣ 代码实现
```python
class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        # 给定一个链表,判定是否有环.
        # 用快慢指针,判定两个指针是否会遇上.如果遇上了就有环.否则就没有环
        p,fp=head,head
        while fp is not None and fp.next is not None:
            p = p.next
            fp = fp.next.next
            if p == fp:
                return True
        return False
```
